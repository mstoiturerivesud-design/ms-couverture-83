# MS Couverture 83 - site GitHub Pages

Site statique prêt pour GitHub Pages.

## Publication rapide

1. Créer un dépôt GitHub, par exemple `ms-couverture-83`.
2. Envoyer tout le contenu de ce dossier dans le dépôt.
3. Dans GitHub : `Settings` -> `Pages` -> `Deploy from a branch`.
4. Choisir la branche `main` et le dossier `/root`.
5. Connecter ensuite le domaine `couvreurvar83.com` dans les réglages GitHub Pages.

## Pages incluses

- `/` : accueil
- `/couvreur-roquebrune-sur-argens/` : qui sommes-nous
- `/prestations-couverture-var/` : prestations
- `/contact/` : contact
- `/page-vierge/` et `/page-vierge-1/` : redirections depuis les anciennes URLs Wix

## Notes

Le formulaire de contact utilise `mailto:` pour rester compatible avec GitHub Pages sans serveur. Pour un vrai formulaire qui arrive directement en boîte mail sans ouvrir le client mail du visiteur, il faudra ajouter un service de formulaire.
